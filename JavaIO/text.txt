Лабораторная работа № 12
Java I/O
(Продолжительность 4 часа)
Цели:
 закрепить знания по технологии Java I/O;
 научиться на практике использовать технологии Java I/O.
ПРАКТИЧЕСКОЕ ЗАДАНИЕ
Задание 1. (10%)
Разработать программу, которая выводит свой код (содержимое *.java файла).
Во время разработки ОБЯЗАТЕЛЬНО средствами nio2:
 определите текущий каталог;
 имя файла;
 перейдите в каталог scr проекта;
 соберите имя файла с текстом программы (добавьте расширение *.java).
Задание 2. (20%)
Разработать программу, которая записывает в типизированный файл целые
числа, затем считывает и рекуррентно рассчитывает среднее.
Задание 3. (20%)
В файл пишутся целые числа. Затем из этого файла, как потока байт,
считывается числа byte, int и float. Произвести расчет считанных элементов при
чтении данных разного типа.
Задание 4. (20%)
Файл задачи 2. Определить количество чисел типа int. Рассчитать среднее
второй половины, третьей четверти.
Задание 5. (20%)
Создайте текстовый файл, в который скопируйте не менее одной страницы
документации по java (можно использовать данное задание). Считывайте
символьный поток пока в третий раз не встретится слово «java».
Задание 6. (10%)
Выполните пример Password из лекции (с. 11). Закомментируйте каждую
строку с пояснением действия.
